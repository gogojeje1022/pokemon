**설치 및 설정 방법**

melonDS 최신 버전을 다운로드합니다.

위 png 파일을 다운로드한 뒤,
안드로이드 기기의 원하는 폴더에 압축 해제합니다.

melonDS를 실행한 후,
Input 설정 → Layouts로 이동하여 + 버튼을 눌러 새 레이아웃을 생성합니다.

Menu → Properties로 이동하여 다음과 같이 설정합니다.

Layout orientation: Portrait

Use default opacity: 비활성화

Layout opacity: 0

설정 후 OK를 누릅니다.

다시 MENU → Backgrounds로 이동합니다.

Portrait background 선택

+ 버튼을 눌러 앞에서 다운로드한 png 파일 선택

이름을 지정하고 OK

생성한 배경을 선택 후 다시 OK

화면 크기를 적절하게 조절한 뒤 뒤,
화면을 드래그하여 오버레이에 맞게 배치합니다.

이동 키, ABXY, Start, Select 버튼을 배경 이미지와 겹치도록 이동

필요 시 버튼 크기 조절

사용하지 않는 버튼은 왼쪽 아래의 DELETE 버튼 눌러서 삭제도 가능합니다.
